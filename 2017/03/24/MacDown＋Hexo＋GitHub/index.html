MacDown＋Hexo＋GitHub搭建个人博客

前言

hexo对Markdown支持，官网提供搭建方式
个人博客最终目的是将Hexo与github page关联起来，需要安装Hexo和git。
为了安装Hexo，我们需要安装node;
为了方便安装git、node，我们需要安装Homebrew；
所以完成这个搭建的目录就定出来了

目录

  • 安装Homebrew 1.1.11-132-g17cc40f
  • 安装git 2.5.4 (Apple Git-61)
  • 安装node v6.3.1
  • 安装hexo 3.2.2
  • 将Hexo与github page关联起来
  • 发布博客
    • 安装MacDown 0.6.4
    • 写博客
    • 发布博客

安装Homebrew

进入Homebrew网站，查看安装Homebrew的指令:

  • 输入指令，安装Homebrew

  • 输入指令，查看版本

安装git

通过Homebrew安装git:

  • 输入指令，安装git

  • 输入指令，查看版本

安装node

通过Homebrew安装node:

  • 输入指令，安装node

  • 输入指令，查看版本

  • 执行brew install node 命令出错

    出错原因:安装文件写入到/usr/local的权限不足
    解决办法:在命令行输入指令，再输入appid的密码，即可授予权限

    (注意：重新打开的命令行窗口，需要重新授予权限。)

安装hexo

进入hexo网站，查看安装hexo的指令:

  • 输入指令，安装hexo

    (注意：在命令行中，对hexo的任何操作，需进入到hexo的初使化目录。上面指令中初使目录是blog。)

  • 输入指令，查看版本

    -bash: npm: command not found

    brew cleanup
    brew link node
    brew uninstall node
    brew install node

    deploy:

    type: git
repository: https://github.com/yourname/yourname.github.io.git
branch: master

    brew cask install macdown


macdown -v


Error: Unknown command: cask


    cd $(brew --repo) 
    git fetch
    git reset --hard origin/master
    brew update
    

    brew cask install macdown
    

hexo new MacDown＋Hexo＋GitHub


hexo d -g
